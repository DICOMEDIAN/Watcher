# Skill: ideation-cycle

## Purpose

Grow a small idea into a connected set of product concepts, user needs, market
needs, and planning artifacts.

## Korean Name

아이데이션 사이클

## Agents

- SMARTIE / 똑똑이: idea gardener, recorder, concept connector, mind-map owner
- MAX / 맥스: market, positioning, strategic expansion
- STEIN / 스타인: safety, compliance, expert-domain challenge
- SENSI / 센시: user perspective, mockup direction, product critique

## Cycle Shape

```text
Seed:
  User gives a short phrase, loose idea, question, or issue.

Expansion:
  MAX, STEIN, SMARTIE, and SENSI brainstorm outward like a snowball.

Recording:
  SMARTIE continuously records new ideas, assumptions, user needs, market needs,
  feature candidates, and contradictions.

Connection:
  SMARTIE groups ideas into conceptual clusters and links them.

Mind Map:
  SMARTIE produces or updates a text mind map that can later become a visual
  artifact.
```

## Inputs

- one-line seed idea
- existing product/project context
- recent Notion discussion
- relevant screenshots or files

## Required Outputs

```markdown
## Seed
## New Ideas
## User Needs
## Market Needs
## Feature Candidates
## Risks And Contradictions
## Concept Clusters
## Mind Map
## Plan Updates
## Next Questions
```

## Iteration Rule

When the user requests N cycles, do not restart from zero. Each round expands
the previous mind map and plan updates.

## SMARTIE Recording Rule

SMARTIE must keep a durable ideation note in the project memory area or Notion
summary area. Raw brainstorming stays in Notion; durable concepts are summarized
into `D:\AI_Hermes` when they become reusable.

## Completion Rule

The cycle is complete when SMARTIE produces:

- expanded concepts
- updated plan notes
- a mind-map section
- clear next questions or next cycle seed
