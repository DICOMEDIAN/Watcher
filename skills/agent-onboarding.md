# Skill: agent-onboarding

## Purpose

Attach a new local agent to Hermes with a clear role, bridge, Notion behavior,
and continuity pack.

## Checklist

1. Register the agent identity.
2. Define role boundaries.
3. Locate the Claude/Codex session or create one.
4. Create a continuity prompt.
5. Create or configure a bridge job.
6. Send a UTF-8 test message.
7. Send an image/file test.
8. Confirm the agent can reply to Notion.
9. Record status in `D:\AI_Hermes\status`.

## Required Agent Fields

- canonical id
- Korean call name
- English call name
- runtime type
- host PC
- project workspace access
- Notion channels
- bridge script
- wake cadence
- reply style
